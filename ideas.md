# Ideias de Design — Portfólio Pedro Andreoli

## Contexto
Pedro é um desenvolvedor Full-Stack com mentalidade de empreendedor. Seu diferencial não é só código — é pensar em vendas, economia de tempo e resultado para o cliente. O portfólio precisa transmitir isso: autoridade, clareza e impacto.

## Ideia 1 — "Terminal Executivo" (Probabilidade: 0.07)

**Design Movement:** Brutalismo Digital + Neo-Noir Tech

**Core Principles:**
- Contraste extremo: preto profundo com verde neon (#00FF88) como acento único
- Tipografia monospace misturada com sans-serif bold para criar tensão visual
- Assimetria intencional: layouts que quebram a grade, criando movimento
- Densidade de informação: dashboards e cards compactos, sem desperdício de espaço

**Color Philosophy:**
- Background: #0A0A0A (quase preto absoluto)
- Surface: #111111 / #1A1A1A
- Accent primário: #00FF88 (verde terminal — remete a código, produtividade, dinheiro)
- Accent secundário: #FF6B35 (laranja queimado — urgência, ação)
- Texto: #E8E8E8 / #A0A0A0

**Layout Paradigm:**
- Sidebar fixa à esquerda com navegação vertical
- Conteúdo principal em grid assimétrico (70/30, 60/40)
- Hero section com texto alinhado à esquerda, sem centralização
- Cards com bordas neon e efeito de "scan line" sutil

**Signature Elements:**
- Cursor piscante ao lado do nome (efeito terminal)
- Linhas de código decorativas em background com baixa opacidade
- Badges de status (ONLINE, DISPONÍVEL) com ponto verde pulsante

**Interaction Philosophy:**
- Hover: bordas acesas com glow neon
- Click: flash rápido de cor
- Scroll: revelação progressiva com fade-in lateral

**Animation:**
- Texto da hero digitado caractere por caractere (typewriter)
- Cards entram deslizando da esquerda com spring
- Números de métricas contam de 0 até o valor final

**Typography System:**
- Display: JetBrains Mono Bold (headings, nome)
- Body: Space Grotesk Regular/Medium
- Accent: JetBrains Mono para labels e badges

## Ideia 2 — "Arquitetura de Negócios" (Probabilidade: 0.08) ← ESCOLHIDA

**Design Movement:** Corporate Dark Premium + Editorial Tech

**Core Principles:**
- Elegância sóbria: dark mode sofisticado que transmite seriedade profissional
- Hierarquia tipográfica agressiva: tamanhos extremos criam impacto visual imediato
- Geometria limpa: bordas finas, espaçamento generoso, alinhamento preciso
- Credibilidade visual: o design precisa fazer o visitante pensar "esse cara é sério"

**Color Philosophy:**
- Background: #0D0F14 (azul-escuro quase preto — profissionalismo, confiança)
- Surface: #161A23 / #1E2330
- Accent primário: #3B82F6 (azul elétrico — tecnologia, confiança)
- Accent secundário: #F59E0B (âmbar/dourado — valor, resultado, empreendedorismo)
- Texto: #F1F5F9 / #94A3B8

**Layout Paradigm:**
- Full-width sections com padding generoso
- Hero assimétrico: texto à esquerda (60%), visual à direita (40%)
- Navegação horizontal fixa no topo com blur backdrop
- Seções alternadas: fundo escuro / fundo levemente mais claro
- Cards com borda sutil e hover elevado

**Signature Elements:**
- Linha dourada fina separando elementos importantes (border-amber)
- Gradiente diagonal sutil no hero (azul escuro → quase preto)
- Tags de tecnologia com fundo translúcido e borda colorida

**Interaction Philosophy:**
- Hover em cards: elevação com sombra azul sutil + borda iluminada
- Botões com preenchimento progressivo no hover
- Links de navegação com underline que cresce da esquerda

**Animation:**
- Hero: palavras entram com stagger de baixo para cima
- Seções: fade-in com translateY ao entrar no viewport
- Números: contagem animada quando visíveis
- Cards de projeto: rotação leve 3D no hover

**Typography System:**
- Display: Sora ExtraBold / Bold (headings principais)
- Body: Sora Regular / Medium
- Mono: JetBrains Mono (código, badges técnicos)
- Hierarquia: 72px / 48px / 32px / 20px / 16px / 14px

## Ideia 3 — "Manifesto Visual" (Probabilidade>0.06)

**Design Movement:** Swiss Modernism + Dark Editorial

**Core Principles:**
- Grid rigoroso com quebras intencionais para criar tensão
- Tipografia como elemento gráfico principal
- Minimalismo funcional: cada elemento tem propósito
- Contraste de escala: elementos gigantes ao lado de elementos minúsculos

**Color Philosophy:**
- Background: #121212
- Accent: #FF3B30 (vermelho — urgência, impacto, ação)
- Texto: #FFFFFF / #666666
- Sem gradientes — apenas cores sólidas

**Layout Paradigm:**
- Grid de 12 colunas com elementos que quebram a grade
- Números de seção gigantes como elementos decorativos
- Texto rotacionado verticalmente como labels de seção

**Signature Elements:**
- Números de seção em 200px como watermark
- Linha vermelha vertical como elemento de identidade
- Citações em destaque com aspas gigantes

**Interaction Philosophy:**
- Inversão de cor no hover (fundo branco, texto preto)
- Transições instantâneas sem easing — cortes diretos

**Animation:**
- Revelação por clip-path (cortina horizontal)
- Sem animações suaves — movimentos rápidos e decisivos

**Typography System:**
- Display: Bebas Neue (headings impactantes)
- Body: IBM Plex Sans
- Contraste extremo de peso: 900 vs 300

## Decisão Final: Ideia 2 — "Arquitetura de Negócios"

Esta abordagem equilibra impacto visual com credibilidade profissional. O dark mode sofisticado com acento azul elétrico e dourado transmite exatamente o que Pedro precisa: um desenvolvedor que pensa como empreendedor, que entrega resultado, e que merece ser contratado.
